Notewithme.com supports pasting images from clipboard. You need to take screenshot to your clipboard before you can paste it:

  • Windows: alt + print screen, ctrl + print screen
  • OS X: command + control + shift +3
  • OS X: command + control + shift +4

After that, just hit paste:

  • Windows: control + v
  • OS X: command + v
